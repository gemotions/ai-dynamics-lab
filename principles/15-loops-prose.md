Negative Emotions create a loop.

There is one Emotion loop for each of the eight Preferences.

Each loop contains three Negative Emotions.

A loop creates pull around its Preference.

The Extrovert loop contains Terror, Dread, and Disbelief.
The Introvert loop contains Creepy, Wrong/Failure, and Rejected.
The Sensor loop contains Scared, Denial, and Guilty.
The Intuitor loop contains Nervous, Stressed, and Sad/Depressed.
The Thinker loop contains Stupid, Anger, and Revenge.
The Feeler loop contains Confused, Crazed, and Gloom.
The Judger loop contains Overwhelmed, Worried, and Ashamed/Judged.
The Perceiver loop contains Dizzy, Despair, and Loss.
